Welcome to the URL Shortening Microservice.

Example usage:

input - creating little url

http://maketiny.herokuapp.com/new/https://www.google.com

output

{"original_url":"https://www.google.com","short_url":"http://maketiny.herokuapp.com/3"}

output (invalid input)

invalid output

then visit the newly created little url

http://maketiny.herokuapp.com/3
